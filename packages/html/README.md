#### Wrangler

To generate using [wrangler](https://github.com/cloudflare/wrangler)

```
npm i
wrangler build
wrangler dev

npm run format
wrangler publish
```

Further documentation for Wrangler can be found [here](https://developers.cloudflare.com/workers/tooling/wrangler).
